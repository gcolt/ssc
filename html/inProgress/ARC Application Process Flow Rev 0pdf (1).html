PINNACLE OF PORT ORANGE 

HOMEOWNERS ASSOCIATION, INC.

 

ARC APPLICATION PROCESS FLOW

Revision: 0                     Date: 08/08/2022    Description: Original Document

 

 

The process to submit an application to the ARC (Architectural Review Committee) is described as follows.

 

General Information

1)    The ARC application form can be obtained from the HOA website or the property manager.

2)    The property manager will maintain correspondence and records for all ARC applications.

3)    The property manager will confirm that all correspondence to the lot owner is received.

4)    In general, an ARC application will be handled as follows.

a)     From the lot owner to the property manager,

b)    From the property manager to the board president,

c)     From the board president to the ARC chairman,

d)    From the ARC chairman to the ARC members.

5)    In general, a response to an ARC application will be handled as follows.

a)     From the ARC chairman to the board president,

b)    From the board president to the property manager,

c)     From the property manager to the lot owner.

6)    An inspection will be performed by an ARC member(s) before and after construction to assure that the completed project conforms to the ARC application and that no damage occurred during construction.

7)    An application that requires a permit from the City of Port Orange will be conditionally approved pending receipt of the permit.

a)     The lot owner must provide a copy of the permit to the property manager before construction begins.

b)    The lot owner must provide a copy of the signed (completed inspection) permit to the property manager after construction is complete.

c)     If heavy equipment needs to be used during construction, a “Performance Assurance Bond” will be required.

8)    Per Article 3, Section 3.3.3 of the subdivision covenants, the board must provide a response made within 45 days of the ARC application.  If a response it not provided within that period, the application is deemed unconditionally approved.

a)     If additional information is needed, the 45-day time period will restart upon receipt of the information.

9)    If construction has not started, an approved application or conditionally approved application will expire after six months from the later of:

a)     The application approval date if no permit is required, or

b)    The permit issue date if required.

10) The lot owner can apply for an extension to an approved application or conditionally approved application by contacting the property manager.

 

Initial Application

11) The lot owner will submit a completed ARC application form, along with all required support information, to the property manager.

12) The property manager will confirm that the lot owner is in good standing before sending the application to the board president.

a)     The lot owner must not have any outstanding debts to the HOA.  This includes, but is not limited to assessments, clubhouse reservation damages, construction damages, etc.

b)    The lot owner must not have any outstanding violations.

13) If the lot owner is in good standing, the property manager will send the application to the board president.

14) The board president will send the application to the ARC chairman.

15) The ARC chairman will send the application to the ARC members.

16) The ARC members will discuss the application and make a recommendation to the board.

17) The board will determine if the application is unconditionally approved, conditionally approved, incomplete, or denied.

 

Unconditionally Approved Application (No Permit Required)

18) The ARC chairman will check the “Approve” box on the application, make applicable comments if needed, and sign it.

19) The ARC chairman will send the approved application to the board president and the ARC members.

20) The board president will send the approved application to the property manager.

21) The property manager will send the approved application to the lot owner.

22) When construction is complete, the lot owner will notify the property manager.

23) The property manager will notify the board president.

24) The board president will notify the ARC chairman.

25) The ARC chairman will notify the ARC members.

26) The ARC member(s) will complete an inspection form, including comments, and send it to the ARC chairman.

27) The ARC chairman will sign the inspection form and make comments as needed.

28) The ARC chairman will send the inspection form to the board president.

29) The board president will send the inspection form to the property manager.

30) If the project is completed per the approved application:

a)     The property manager will notify the lot owner that the application is closed. 31) If the project is not completed per the approved application:

a)     The board president will meet with the other board members to discuss the inspection report.

b)    The board members will determine how to proceed.

c)     The board president will provide guidance to the property manager.

d)    The property manager will notify the lot owner.

e)     The lot owner must respond to the property manager within 14 days.

f)     The property manager will send the response to the board president.

g)    The board president will meet with the other board members to discuss the response.

h)    The board members will determine how to proceed.

i)      The board president will provide guidance to the property manager.

j)      The property manager will notify the lot owner.

k)    The process will continue until a resolution is achieved.

l)      If a response is not submitted within the required time frame, the property manager will issue a violation notice to the lot owner and send the violation notice to the board president.

 

Conditionally Approved Application (Permit Required)

32) The ARC chairman will check the “Approve as Noted” box on the application, make applicable comments if needed, and sign it.

a)     If heavy equipment is to be used, the lot owner will be required to submit a

“Performance Assurance Bond” to cover any damage that occurs during construction as described in the application.  The bond must be submitted to the property manager before construction begins.

b)    The ARC chairman will make a note on the application that a bond is required.

33) The ARC chairman will send the approved application to the board president and the ARC members.

34) The board president will send the approved application to the property manager.

35) The property manager will send the approved application to the lot owner.

a)     If needed, the property manager will inform the lot owner that a bond is required.

36) The lot owner will submit the project plans to the City of Port Orange to obtain a permit.

37) The lot owner will provide a copy of the permit to the property manager upon receipt.

a)     The property manager will add the permit number and issue date to the approved application.

b)    If needed, the lot owner will submit a check for the bond to the property manager.

38) The property manager will send the permit to the board president.

39) The board president will send the permit to the ARC chairman.

40) When construction is complete, the lot owner will send a copy of the permit inspection complete (signed) to the property manager.

a)     The property manager will add the permit inspection complete date to the approved application.

41) The property manager will notify the board president.

42) The board president will notify the ARC chairman.

43) The ARC chairman will notify the ARC members.

44) The ARC member(s) will complete an inspection form, including comments, and send it to the ARC chairman.

45) The ARC chairman will sign the inspection form and make comments as needed.

46) The ARC chairman will send the inspection form to the board president.

47) The board president will send the inspection form to the property manager.

48) If the project is completed per the approved application and no damage has occurred:

a)     The property manager will notify the lot owner that the application is closed and return the check for the bond to the lot owner.

49) If the project is not completed per the approved application:

a)     The board president will meet with the other board members to discuss the inspection report.

b)    The board members will determine how to proceed.

c)     The board president will provide guidance to the property manager.

d)    The property manager will notify the lot owner.

e)     The lot owner must respond to the property manager within 14 days.

f)     The property manager will send the response to the board president.

g)    The board president will meet with the other board members to discuss the response.

h)    The board members will determine how to proceed.

i)      The board president will provide guidance to the property manager.

j)      The property manager will notify the lot owner.

k)    The process will continue until a resolution is achieved.

l)      If a response is not submitted within the required time frame, the property manager will issue a violation notice to the lot owner and send the violation notice to the board president.

50) If damage occurs during construction:

a)     The property manager will inform the lot owner that damage has occurred and repairs will be made using the funds from the bond.  Any remaining balance will be returned to the lot owner by the property manager.

b)    If the funds from the bond are insufficient, the property manager will send a bill for the excess repair costs to the lot owner.  Payment is due within 14 days.

 

Incomplete Application

51) The ARC chairman will check the “Incomplete as Noted” box on the application, make applicable comments if needed, and sign it.

52) The ARC chairman will send the incomplete application to the board president.

53) The board president will send the incomplete application to the property manager.

54) The property manager will send the incomplete application to the lot owner and request the required information.

55) The lot owner will provide the required information to the property manager.

a)     If the lot owner does not provide the required information within 14 days, the application will expire.

 

Denied Application

56) The ARC chairman will check the “Deny” box on the application, make applicable comments if needed, and sign it.

57) The ARC chairman will send the denied application to the board president.

58) The board president will send the denied application to the property manager. 59) The property manager will send the denied application to the lot owner.

 

Denied Application Appeal Process

60) The lot owner must notify the property manager of the intention to appeal the denied application.

61) The property manager will send the appeal notice to the board president.

62) The board president will add the appeal to the agenda of the next board meeting.

63) The board president will inform the property manager of the board meeting location, date, and time.

64) The property manager will inform the lot owner that the appeal will be heard at the next board meeting.

a)     If the board meeting time is not acceptable, the appeal will be added to the agenda of upcoming board meetings for up to three times.

65) The property owner will attend the board meeting and appeal the denied application.

66) The board will determine how to proceed at the board meeting and provide guidance to the lot owner.

67) If the lot owner cannot attend any of the board meetings, the lot owner can submit written documentation for the appeal to the property manager.

a)     The property manager will send the documentation to the board president.

b)    The board president will send the documentation to the board members.

c)     The board members will determine how to proceed.

d)    The board president will provide guidance to the property manager.

e)     The property manager will notify the lot owner.

 

Extended Application

68) The lot owner must request an extension from the property manager within the sixmonth time period for the approved application time.  The request must include the reason for the extension.

69) The property manager will send the request to the board president.

70) The board president will meet with the other board members to discuss the extension.

71) The board members will determine how to proceed.

72) The board president will provide guidance to the property manager.

73) The property manager will notify the lot owner of the decision by the board.

a)     If the extension is granted, the application will expire within the next six-month time period.  No additional extensions will be granted.  As such, the lot owner will be required to submit another application for the project.

b)    If the extension is not granted, the lot owner may appeal the decision.  The appeal process for an extension will be the same as the appeal process for a denied application as described previously.
